# Netflix Content Analysis

Exploratory data analysis of Netflix's content library using Python and pandas.

## Questions Answered
- Which countries produce the most Netflix content?
- How does production split between Movies and TV Shows by country?
- How has Netflix's content volume grown year over year?

## Key Findings
- The USA leads with 3600+ titles, over 3x second-place India
- South Korea's content is 75% TV Shows — driven by Korean dramas
- Netflix grew 472% in 2016, marking its major content expansion push

## Dataset
Netflix Movies and TV Shows — available on Kaggle
